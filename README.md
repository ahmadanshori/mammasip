# mammasip

aplikasi kedokteran

API

```groovy
app => api
```

image, icon, font

```groovy
app => assets
```

Routing

```groovy
app => config => routes
```

global Color, Font, Size

```groovy
app => constants => theme
```

global Function ex: format rupiah

```groovy
app => libs => theme
```

global Hooks ex: handleError

```groovy
app => hooks
```
